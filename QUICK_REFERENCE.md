## 📱 Web 应用功能

| 功能 | 按钮 | 说明 |
|-----|------|------|
| 路径输入 | 📁 输入路径 | 输入或选择音乐目录 |
| 上一级 | ⬆️ 上一级 | 返回上级目录 |
| 子目录 | 📂 子目录名 | 快速跳转到子目录 |
| 扫描 | 🔍 开始扫描 | 扫描音乐库 |
| 重复检测 | 🔁 重复歌曲 (N) | 查看重复文件 |
| 版本分析 | 🎚️ 多版本 (N) | 查看多格式歌曲 |
| 低质检测 | 🎧 仅 MP3 (N) | 查看需要升级的歌曲 |
| 导出清单 | 📝 导出清单 | 生成下载清单 |
| 删除 | 🗑️ 删除 | 删除标记的文件 |

---

## 📊 统计指标

| 指标 | 含义 | 目标 |
|-----|------|------|
| 📦 文件总数 | 音乐库中所有文件 | 越少越好 |
| 🎵 唯一歌曲 | 去重后的歌曲数 | 即使下降不大也正常 |
| 🎚️ 多版本歌曲 | 有多个格式的歌曲 | 逐步降低 |
| 🎧 仅 MP3 歌曲 | 低质量歌曲 | 逐步升级到 0 |

## 🎯 使用场景速查

### 场景 1：快速清理重复
```
扫描 → 🔁 重复歌曲 → 🗑️ 删除 → ✅

```

```
导出 CSV → Excel 分析 → 生成报告 → ✅
```

## 🔧 快速问题排查

| 问题 | 解决方案 |
|-----|---------|
| 扫描慢 | 关闭其他应用，检查磁盘速度 |
| 找不到文件 | 检查路径是否存在，是否有权限 |
| 删除失败 | 关闭播放器，检查文件是否被占用 |
| 导出失败 | 确保 exports 目录存在，有写权限 |
| 应用崩溃 | 查看控制台错误信息，重启应用 |

## 🎉 最佳实践

✅ **推荐做法**
- 定期（每月）扫描和维护
- 下载前先导出清单
- 优先下载 FLAC 格式
- 保留备份副本
- 逐步升级而不是一次性转换

❌ **避免做法**
- 未备份前删除大量文件
- 同时执行多个任务
- 直接修改数据库文件
- 删除正在播放的文件
- 跳过验证步骤
